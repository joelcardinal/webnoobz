Key Commands
Cycle Editors:    Ctrl+j
Update Render:    Update button or Ctrl+k
Resize Workspace: Ctrl+d
Resize Render:    Ctrl+f
Features
Toggles viewing of collaborate dock. On by default.
Toggles locking pupils' editors, so they can not edit. Off by default.
Toggles sync of editors, when you switch they all switch. On by default.

Webnoobz is an online code editor with real-time collaboration, suitable for teaching client-side web development.

For more information visit Webnoobz on GitHub.
